# 发布就绪门禁清单（Go/No-Go）

## 1. 功能门禁
1. REQ-001 ~ REQ-020 映射实现完成率 >= 95%。
2. 核心链路 E2E 用例通过率 >= 95%。
3. RepairTask 状态机核心迁移与失败迁移用例全通过。

## 2. 质量门禁
1. 关键模块单测覆盖率 >= 80%。
2. 无 P0/P1 未关闭缺陷。
3. lint/test/scan 全通过。

## 3. 稳定性门禁
1. 压测达到 2,000 events/min 持续 60 分钟。
2. 高峰后积压在 60 分钟内清空。
3. 关键 SLO 不突破阈值。

## 4. 成本门禁
1. 日预算熔断可触发并恢复。
2. 单任务预算熔断可触发并审计。
3. 模型路由命中率与成本看板可观测。

## 5. 运维门禁
1. Oncall 手册可执行演练通过。
2. DR 演练满足 RTO <= 30min，RPO <= 5min。
3. 变更管理流程已启用并可追溯。

## 6. 发布决策
- 全部门禁通过：Go
- 任一 P0 门禁失败：No-Go
- P1 门禁失败：需架构负责人/SRE负责人联合签署例外放行
