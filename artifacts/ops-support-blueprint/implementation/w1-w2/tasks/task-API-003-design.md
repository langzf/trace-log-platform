## 变更目标
实现 issue 列表查询能力，支持按项目、状态、来源过滤，并支持 `limit` 限流返回，供接收器控制台和后续执行器编排读取待处理问题。

## 变更范围
- In scope:
  - 新增 `GET /v1/issues` 路由
  - `projectKey/status/sourceType/limit` 查询参数解析
  - 结果按 `updatedAt` 倒序返回
- Out of scope:
  - 游标分页与总量统计分段查询
  - 复杂组合检索（全文检索、时间窗口）

## 风险与回退
- 风险：默认分页参数解析异常可能导致空列表误判
- 回退方案：回退到仅内部存储调用方式，暂时下线接口查询
