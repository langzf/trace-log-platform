# Error Code Specification

| Error Code | HTTP | 含义 | 处理建议 |
|---|---:|---|---|
| ERR-1001 | 400 | 请求参数无效 | 检查 schema 与必填字段 |
| ERR-1002 | 400 | 幂等键冲突且内容不一致 | 使用新的 Idempotency-Key |
| ERR-1003 | 404 | 资源不存在 | 校验 ID 与项目边界 |
| ERR-1004 | 409 | 状态机冲突（非法状态迁移） | 读取最新状态后重试 |
| ERR-1005 | 422 | 置信度不足，拒绝自动修复 | 转人工审批或升级模型 |
| ERR-1006 | 429 | 预算或并发超限 | 降低并发/等待窗口 |
| ERR-1007 | 500 | 内部处理失败 | 查看审计与任务日志 |
| ERR-1008 | 502 | 执行器调用失败 | 切换执行器或重试 |
| ERR-1009 | 504 | 执行超时 | 调整 timeout 或拆分任务 |
| ERR-1010 | 412 | 质量门禁未通过 | 修复测试失败后重试 |
| ERR-1011 | 423 | 发布锁定中 | 等待发布窗口或人工解锁 |
| ERR-1012 | 503 | 系统熔断中 | 等待恢复后重试 |
